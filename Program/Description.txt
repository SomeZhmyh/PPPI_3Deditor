1.	Файл:
Создать новый проект;
Открыть существующий проект;
Сохранить проект.
2.	Камера:
Сдвинуть;
Приблизить/отдалить;
Повернуть;
Закрепить на объекте.
3.	Редактировать:
Выбрать;
Вырезать/копировать/вставить;
Отменить/повторить действие;
Сгруппировать объекты;
Удалить/скрыть;
Редактировать пивот (якорь).
4.	Режимы выделения:
Привязка к точкам;
Привязка к граням;
Привязка к полигонам.
5.	Инструменты редактирования объектов:
Переместить;
Повернуть;
Масштабировать;
Снять фаску;
Выдавить;
Добавить сечение.
6.	Инструменты создания объектов:
Создать точку;
Создать фигуру;
Создать кривую;
Создать анимацию;
Добавить риггинг (кость для анимации).
7.	Режимы отображения:
Каркасное отображение;
Отображение текстур;
Сглаживание.
